## Ana Sayfaya Ürün Ekleme Planı

### Sorun
`ProductShowcase.tsx` dosyasındaki sorgu, "Çok Satanlar" ve "Yeni Gelenler" sekmeleri için veritabanında `is_featured = true` ve `is_new = true` olan ürünleri filtreliyor. Eğer veritabanında bu bayraklarla işaretlenmiş ürün yoksa, ana sayfada hiç ürün görünmüyor.

### Çözüm
Filtreleri kaldırarak tüm ürünleri göstereceğim:

1. **Satır 255-261**: `bestsellers` ve `new` sekmeleri için filtreleri kaldır
   - `query.eq("is_featured", true)` yerine hiçbir filtre uygulama
   - `query.eq("is_new", true)` yerine hiçbir filtre uygulama

2. **Sonuç**: Tüm ürünler ana sayfada görünecek

### Değişiklik
```typescript
// Önce (Satır 255-261):
switch (activeTab) {
  case "bestsellers":
    query = query.eq("is_featured", true);
    break;
  case "new":
    query = query.eq("is_new", true);
    break;
}

// Sonra:
switch (activeTab) {
  case "bestsellers":
  case "new":
    // Tüm ürünleri göster - filtre yok
    break;
}
```
