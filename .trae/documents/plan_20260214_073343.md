## Admin Panel Menü Yüklenme Sorunu Düzeltme Planı

### Sorunun Kaynağı:

1. **Role kontrolü hatası** (AdminSidebar.tsx, satır 192-197):
   - `if (!role) return false;` - role null olduğunda tüm menü öğeleri filtreleniyor
   - Kullanıcı giriş yapmış ama profil veritabanında yoksa menü boş görünüyor

2. **Menü öğeleri loading sırasında gösterilmiyor**:
   - Loading state geçtikten sonra role null ise menü boş kalıyor

### Yapılacak Değişiklikler:

```tsx
// AdminSidebar.tsx - filteredItems düzeltmesi:

// Önceki (sorunlu):
const filteredItems = MENU_ITEMS.filter(item => {
  if (!role) return false;
  if (role === 'super_admin') return true;
  if (item.title === "Ana Sayfa") return true;
  return hasPermission(role as UserRole, item.href);
});

// Yeni (düzeltilmiş):
// Loading sırasında tüm menüyü göster, role yüklendikten sonra filtrele
const filteredItems = loading 
  ? MENU_ITEMS  // Loading sırasında tümünü göster
  : MENU_ITEMS.filter(item => {
      if (!role) return true; // Role yoksa tümünü göster
      if (role === 'super_admin') return true;
      if (item.title === "Ana Sayfa") return true;
      return hasPermission(role as UserRole, item.href);
    });
```
