## Admin Panel Yüklenme Sorunu Düzeltme Planı

### Sorunun Kaynağı:
1. AdminSidebar Supabase'den kullanıcı bilgisi çekiyor (timeout olabilir)
2. Mobil kontrolü `window` kullanıyor (SSR sorunu olabilir)
3. Loading state için timeout yok

### Yapılacak Değişiklikler:

1. **AdminSidebar.tsx** - Geliştirmeler:
   - Supabase çağrısına timeout ekle (5 saniye)
   - Hata durumunda loading'i false yap
   - Mobil kontrolü için `useMediaQuery` hook kullan
   - Window erişimini güvenli hale getir

2. **AdminLayout** - İyileştirmeler:
   - Daha iyi loading state
   - Hata durumunda kullanıcıyı yönlendir

```tsx
// AdminSidebar'da timeout örneği:
useEffect(() => {
  async function getUser() {
    const timeout = setTimeout(() => setLoading(false), 5000);
    
    try {
      const { data: { user } } = await supabase.auth.getUser();
      // ... user processing
    } catch (error) {
      console.error(error);
    } finally {
      clearTimeout(timeout);
      setLoading(false);
    }
  }
  getUser();
}, []);
```
