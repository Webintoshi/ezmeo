## Site Hızı Optimizasyonu

Site hızı, sadece SEO ranking faktörü değil, doğrudan conversion oranını etkileyen en kritik metriklerden biridir. Araştırmalar şunu gösteriyor:

- **1 saniye** yükleme süresi: %90 conversion
- **3 saniye** yükleme süresi: %32 conversion düşüşü
- **5 saniye** yükleme süresi: %90 bounce artışı

Bu rehberde, pratik ve hemen uygulanabilebilir optimizasyon tekniklerini ele alacağız.

## Resim Optimizasyonu (En Etkili Quick Win)

Resimler, genellikle sayfanın toplam boyutunun **%50-70'ini** oluşturur.

### 1. Format Seçimi

| Format | Kullanım | Boyut Tasarrufu |
|---|---|---|
| **WebP** | Fotoğraflar, grafikler | %25-35 vs JPEG |
| **AVIF** | Modern browser'lar | %50 vs JPEG |
| **PNG** | Transparent gerekli | WebP'e tercih etme |
| **SVG** | Icons, logos | Lossless, scaleable |

```html
<!-- Next.js ile automatic WebP -->
import Image from 'next/image';

<Image
  src="/hero.jpg"
  alt="Hero"
  width={1920}
  height={1080}
  // Otomatik WebP conversion
/>
```

### 2. Lazy Loading

```javascript
// ✅ Native lazy loading (Chrome 76+)
<img src="image.jpg" loading="lazy" alt="..." />

// ✅ Next.js Image component
<Image
  src="/product.jpg"
  loading="lazy"
  placeholder="blur" // Blur placeholder
/>
```

### 3. Responsive Images

```html
<img
  srcset="small.jpg 500w,
          medium.jpg 1000w,
          large.jpg 2000w"
  sizes="(max-width: 600px) 500px,
         (max-width: 1200px) 1000px,
         2000px"
  src="large.jpg"
  alt="Responsive image"
/>
```

## CDN (Content Delivery Network)

CDN, statik dosyaları kullanıcıya en yakın sunucudan sunar.

**Faydaları:**
- 100-200 ms latency azaltma
- Düşük bandwidth maliyeti
- Yüksek availability
- DDoS koruması

**Popüler CDN'ler:**
- Cloudflare (Ücretsiz plan)
- AWS CloudFront
- Fastly
- KeyCDN

```javascript
// Next.js ile CDN yapılandırması
// next.config.js
module.exports = {
  images: {
    domains: ['cdn.example.com'],
    loader: 'custom',
    loaderFile: './lib/image-loader.js',
  },
};
```

## Browser Caching

Returning visitors için kritik.

```apache
# .htaccess ile cache headers
<IfModule mod_expires.c>
  ExpiresActive On

  # Resimler: 1 yıl
  ExpiresByType image/jpeg "access plus 1 year"
  ExpiresByType image/png "access plus 1 year"

  # CSS/JS: 1 ay
  ExpiresByType text/css "access plus 1 month"
  ExpiresByType application/javascript "access plus 1 month"

  # HTML: Cache yok
  ExpiresByType text/html "access plus 0 seconds"
</IfModule>
```

```javascript
// Next.js ile cache headers
// next.config.js
module.exports = {
  async headers() {
    return [
      {
        source: '/images/:path*',
        headers: [
          {
            key: 'Cache-Control',
            value: 'public, max-age=31536000, immutable',
          },
        ],
      },
    ];
  },
};
```

## Server-Side Optimizasyonlar

### Gzip / Brotli Compression

```javascript
// Next.js ile compression

  compress: true, // Varsayılan true

```

### HTTP/2

```nginx

server {
    listen 443 ssl http2;

    ssl_certificate /path/to/cert.pem;
    ssl_certificate_key /path/to/key.pem;
}
```

## Ön Yükleme Stratejileri

### Resource Hints

```html
<head>

  <!-- Bağlantı kur -->
  <link rel="preconnect" href="https://cdn.example.com">

  <!-- Kritik kaynakları önceden yükle -->
  <link rel="preload" href="/font.woff2" as="font" crossorigin>

</head>
```
