## Teknik SEO Nedir?

Teknik SEO, web sitenizin arama motorları tarafından daha iyi anlaşılması, taranması ve dizine eklenmesi için yapılan optimizasyonları içerir. İçerik ve off-page SEO'dan farklı olarak, teknik SEO tamamen sitenizin altyapısı ve performansıyla ilgilidir.

2026'da teknik SEO, sıradan meta tag'lerden çok daha fazlasını içeriyor. Google'ın Page Experience güncellemesiyle birlikte, **Core Web Vitals** arama sıralamalarında kritik bir faktör haline geldi.

### Neden Teknik SEO Önemli?

- **Taranabilirlik:** Arama motorlarının sitenizi keşfetmesini sağlar
- **İndeksleme:** İçeriklerinizin veritabanına eklenmesini garantiler
- **Performans:** Sayfa hızı kullanıcı deneyimini doğrudan etkiler
- **Mobil Uyumluluk:** Mobile-first indexing ile zorunlu hale geldi
- **HTTPS:** Güvenlik sinyali ve ranking faktörü

## Teknik SEO Bileşenleri

### 1. Site Hızı ve Performans

Sayfa yükleme hızı, hem ranking faktörü hem de conversion oranı üzerinde doğrudan etkiye sahiptir. Araştırmalar şunu gösteriyor:

- 1 saniyeden yüklenen siteler %90 conversion oranına sahip
- 5 saniyeden uzun süren sitelerde conversion %38'e düşüyor
- Her 100 ms gecikme, %1 conversion kaybına neden oluyor

**Optimizasyon Stratejileri:**
- Resim optimizasyonu (WebP formatı, lazy loading)
- CSS/JS minification ve compression
- CDN kullanımı
- Browser caching
- Server response time optimizasyonu

### 2. Core Web Vitals

Google'ın resmi page experience metrikleri:

#### LCP (Largest Contentful Paint)
- **Hedef:** < 2.5 saniye
- **Ne ölçer:** Ana içeriğin yüklenme süresi
- **İyileştirme:** Resim optimizasyonu, pre-fetching

#### INP (Interaction to Next Paint)
- **Hedef:** < 200 ms
- **Ne ölçer:** Kullanıcı etkileşimi sonrası responsiveness
- **İyileştirme:** JavaScript execution time azaltma

#### CLS (Cumulative Layout Shift)
- **Hedef:** < 0.1
- **Ne ölçer:** Görsel stabilite
- **İyileştirme:** Resim yer ayırma, font yükleme stratejileri

### 3. Taranabilirlik (Crawlability)

Arama motorlarının sitenizi keşfetmesi için:

- **robots.txt:** Botlara rehberlik eder
- **XML Sitemap:** Tüm önemli sayfaları listeler
- **Canonical Tags:** Duplicate content'i önler
- **Noindex:** Düşük değerli sayfaları gizler

### 4. Site Mimarisi

İyi bir site yapısı:

- **Hierarchy:** Ana sayfa → Kategoriler → Ürünler
- **URL Structure:** Temiz ve keyword'li URL'ler
- **Navigation:** Mantıklı internal linking
- **Breadcrumb:** Hem UX hem SEO için gerekli

## Teknik SEO Kontrol Listesi 2026

### Zorunlu (P0)
- [ ] HTTPS sertifikası aktif
- [ ] Mobile-friendly tasarım
- [ ] Core Web Vitals "Good" seviyesinde
- [ ] XML sitemap oluşturulmuş
- [ ] robots.txt yapılandırılmış
- [ ] Structured data (Schema.org) eklenmiş

### Önemli (P1)
- [ ] Canonical tags kullanılıyor
- [ ] 404 sayfaları yönlendirilmiş
- [ ] Redirect chain'ler yok
- [ ] Meta descriptions optimize edilmiş
- [ ] H tag'leri hiyerarşik

### İyileştirme (P2)
- [ ] Breadcrumb navigation
- [ ] Pagination optimizasyonu
- [ ] Faceted navigation handling
- [ ] Infinite scroll pagination
- [ ] International SEO (hreflang)
