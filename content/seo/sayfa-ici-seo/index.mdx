## Sayfa İçi SEO (On-Page SEO) Nedir?

On-page SEO, web sitenizin her bir sayfasının arama motorları için optimize edilmesidir. Off-page SEO (backlink'ler) ve teknik SEO'dan farklı olarak, on-page SEO **tamamen sizin kontrolünüzdedir.**

2026'da on-page SEO, artık sadece keyword stuffing değil. Google'ın BERT ve MUM algoritmalarıyla birlikte, **arama niyeti (search intent)** ve **içerik kalitesi** ön planda.

## On-Page SEO Bileşenleri

### 1. Anahtar Kelime Araştırması

Doğru anahtar kelimeleri seçmek, başarının yarısıdır.

**Araçlar:**
- Ahrefs / Semrush (ücretli)
- Google Keyword Planner (ücretsiz)
- AnswerThePublic (sorular için)
- Google Search Console (mevcut performans)

**Strateji:**
- Short-tail (1-2 kelime): Yüksek hacim, yüksek rekabet
- Long-tail (3+ kelime): Düşük hacim, düşük rekabet
- Question keywords: Voice search için kritik

### 2. Başlık Etiketleri (H1-H6)

```html
<!-- ❌ Yanlış -->
<h1>Selam</h1>
<h3>Core Web Vitals</h3>

<!-- ✅ Doğru -->
<h1>Core Web Vitals Nedir? 2026 Kapsamlı Rehber</h1>
<h2>LCP: Largest Contentful Paint</h2>
<h3>Hedef Değerler</h3>
```

**Kurallar:**
- Sayfada sadece **1 tane H1** olsun
- H1'de primary keyword olsun
- H2-H6 hiyerarşik olsun (atlamalar yok)
- Her heading açıklayıcı olsun

### 3. Meta Tags

**Title Tag:**
- 50-60 karakter (Google truncates after 60)
- Primary keyword başta olsun
- Brand name sona (opsiyonel)
- Unique olsun (her sayfa için farklı)

```
✅ Core Web Vitals Nedir? 2026 Rehberi | Ezmeo SEO
❌ 10 SEO Rehberleri - Ezmeo
```

**Meta Description:**
- 150-160 karakter
- CTA (call-to-action) içersin
- Keyword'ler doğal şekilde
- Copy-paste edilir gibi olsun

```
✅ Core Web Vitals metriklerini öğrenin. LCP, INP, CLS nasıl
iyileştirilir? Ücretsiz rehberimizi okuyun ve hemen optimize etmeye başlayın!
```

### 4. URL Yapısı

```bash
✅ /seo/teknik-seo/core-web-vitals
✅ /seo/core-web-vitals-nedir

❌ /seo/post?id=12345
❌ /seo/2024/02/19/core-web-vitals-article-v2
```

**Best Practices:**
- Short (≤ 3-4 path segments)
- Keyword içersin
- Lowercase, hyphen-separated
- Static (dynamic params avoid)

### 5. İçerik Optimizasyonu

**Semantik SEO:**
- LSI keywords (latent semantic indexing)
- Related entities kullan
- TF-IDF (term frequency-inverse document frequency) analiz et

**Content Length:**
- Pillar sayfası: 2,000-4,000 kelime
- Cluster sayfası: 1,500-3,000 kelime
- Blog post: 1,000-2,000 kelime

## Search Intent (Arama Niyeti)

Her sorgunun bir niyeti vardır. Bunu anlamak kritik.

| Niyet | Açıklama | İçerik Türü |
|---|---|---|
| **Informational** | Öğrenmek istiyor | Rehber, how-to, tutorial |
| **Navigational** | Specific site arıyor | Home page, login |
| **Commercial** | Karar aşamasında | Karşılaştırma, review |
| **Transactional** | Satın almak istiyor | Product page, checkout |

**Örnek:**
- "seo nedir" → Informational (rehber)
- "ahrefs login" → Navigational (login page)
- "ahrefs vs semrush" → Commercial (comparison)
- "ahrefs pricing" → Transactional (pricing page)

## On-Page SEO Checklist 2026

- [ ] Title tag unique ve optimize edilmiş
- [ ] Meta description unique ve CTA'lı
- [ ] H1 tek, H2-H6 hiyerarşik
- [ ] URL short, keyword'li, static
- [ ] Content search intent'e uygun
- [ ] Internal links (min 3 outbound)
- [ ] External authoritative links (1-2)
- [ ] Image alt text tüm resimlerde
- [ ] Schema.org JSON-LD eklenmiş
- [ ] Mobile-responsive
- [ ] Page speed (LCP < 2.5s)
- [ ] Content fresh (3 ay içinde güncelle)
