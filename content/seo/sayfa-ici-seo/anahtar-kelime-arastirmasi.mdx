## Anahtar Kelime Araştırması Nedir?

Anahtar kelime araştırması (keyword research), hedef kitlenizin arama motorlarına ne yazdığını bulma sürecidir. Bu, SEO stratejisinin **temel taşıdır.**

Yanlış anahtar kelimelerle çalışırsanız:
- Ayda 10 organik ziyaretçi alırsınız
- Conversion oranı %0.1 olur
- 6 ay boyunca sıralama beklersiniz

Doğru anahtar kelimelerle:
- Ayda 10.000+ organik ziyaretçi
- Conversion oranı %5+
- 3-6 ayda sıralama

## Anahtar Kelime Türleri

### 1. Short-Tail (Head Keywords)

**Özellikler:**
- 1-2 kelime
- Yüksek search volume (10K+/ay)
- Çok yüksek rekabet
- Düşük conversion oranı

**Örnekler:**
- "SEO"
- "E-ticaret"
- "Dijital pazarlama"

**Strateji:** Geniş markalar için uygundur, küçük sitelerden uzak durun.

### 2. Mid-Tail (Body Keywords)

**Özellikler:**
- 2-3 kelime
- Orta search volume (1K-10K/ay)
- Orta rekabet
- İyi conversion

**Örnekler:**
- "Teknik SEO nedir"
- "E-ticaret SEO stratejileri"
- "Dijital pazarlama kursu"

**Strateji:** Çoğu site için ideal.

### 3. Long-Tail Keywords

**Özellikler:**
- 4+ kelime
- Düşük search volume (<1K/ay)
- Düşük rekabet
- Yüksek conversion oranı

**Örnekler:**
- "Core Web Vitals nasıl iyileştirilir"
- "E-ticaret sitesi için SEO checklist"
- "Dijital pazarlama ajansı İstanbul fiyatları"

**Strateji:** Tüm siteler için, özellikle yeni siteler.

## Anahtar Kelime Araştırma Süreci

### Faz 1: Brainstorming (1 saat)

1. **Seed keywords** belirle
   - Industry terms
   - Product/service names
   - Problem statements

2. **Customer questions**
   - Sales team'den soruları al
   - Support tickets'ı incele
   - Social media comments

3. **Competitor analysis**
   - Competitor'ların keywords
   - Title tags, H1'ler
   - Blog topics

### Faz 2: Data Collection (2-4 saat)

```javascript
// Keyword research checklist
const keywordMetrics = {
  term: "core web vitals",
  volume: 1200,           // Aylık arama hacmi
  difficulty: 45,         // 0-100 scale
  cpc: 2.50,              // Cost per click
  competition: 0.6,       // AdWords competition
  intent: "informational", // Arama niyeti
  features: {             // SERP features
    featuredSnippet: true,
    peopleAlsoAsk: true,
    knowledgePanel: false
  }
};
```

### Faz 3: Filtering (1 saat)

**Kriterler:**
- Minimum search volume: 100/ay
- Maximum difficulty: 70 (yeni siteler için 40)
- Relevance: 8/10+
- Intent match: Business goal'larla uyumlu

### Faz 4: Clustering (1 saat)

```javascript
// Keyword clustering
const clusters = {

    "cls fix"
  ]
};
```

## Search Intent Analysis

Her keyword'ün bir niyeti vardır ve bunu anlamak kritik.

| Intent | Açıklama | Content Type |
|---|---|---|
| **Informational** | Öğrenmek istiyor | Rehber, how-to |
| **Navigational** | Specific site | Home, login |
| **Commercial** | Karar aşaması | Comparison, review |
| **Transactional** | Satın almak | Product page |

**Neden Önemli?**

```bash
# Aynı keyword, farklı niyetler
"seo" → Informational (recherche)
"ahrefs seo" → Navigational (Ahrefs site)
"ahrefs vs semrush" → Commercial (comparison)
"ahrefs pricing" → Transactional (pricing)
```

Google, intent mismatch'ı cezalandırır.

## Keyword Difficulty (KD) Nasıl Yorumlanır?

| KD Skoru | Rekabet | Strateji |
|---|---|---|
| 0-20 | Çok düşük | Hemen hedefle |
| 21-40 | Düşük | 3-6 ayda sıralama |
| 41-60 | Orta | 6-12 ay, quality content gerekli |
| 61-80 | Yüksek | Otorite gerekli, long road |
| 81-100 | Çok yüksek | Büyük markalar için |

**Not:** KD sadece tahmindir. Real competition, SERP'i analiz ederek anlaşılır.

```javascript
// Real competition checklist
const competitionAnalysis = {
  top10: [
    { domain: "ahrefs.com", dr: 90, ur: 70 },
    { domain: "moz.com", dr: 92, ur: 75 },
    { domain: "small-blog.com", dr: 30, ur: 15 } // Opportunity!
  ],
  avgDR: 70,        // Sitenizin DR'ı 40 ise, zor
  avgUR: 50,        // Page-level authority
  backlinks: 1000,  // Ortalama backlink sayısı
  canWin: false     // recommendation
};
```

## Long-Tail Stratejisi

### Neden Long-Tail?

1. **Düşük Rekabet:** Büyük siteler ilgilenmez
2. **Yüksek Conversion:** Specific intent
3. **Hızlı Sıralama:** 1-3 ayda ranking
4. **Voice Search:** Mobil aramalar için uygun

### Long-Tail Bulma

**Method 1: Google Autofill**
```
"seo " →
"seo nedir"
"seo nasıl yapılır"
"seo analizi nasıl yapılır"
```

**Method 2: People Also Ask**
```
Google'da ara → "SEO nedir"
→ SERP'de "People also ask" section
→ Tüm soruları topla
```

**Method 3: Question Keywords**
```
AnswerThePublic → "SEO"
→ What, how, why, when, where questions
→ Her question long-tail keyword
```

## Keyword Mapping

Her keyword'ü bir sayfaya atayın.

```javascript
// Keyword mapping
const keywordMap = {
  pillar: {
    keyword: "teknik seo",
    url: "/seo/teknik-seo",
    type: "pillar"
  },
  clusters: [
    {
      keyword: "core web vitals nedir",
      url: "/seo/teknik-seo/core-web-vitals",
      type: "cluster",
      priority: "high"
    },
    {
      keyword: "site hızı optimizasyonu",
      url: "/seo/teknik-seo/site-hizi",
      type: "cluster",
      priority: "medium"
    }
  ]
};
```

**Rules:**
- 1 keyword = 1 page (iki sayfa aynı keyword'ü hedeflemesin)
- Similar keywords → merge (canonical veya redirect)
- Primary keyword in URL, H1
- Secondary keywords in content
