# Odeme Saglayici Logolari

Bu klasore `/admin/ayarlar/odeme/yeni` sayfasinda kullanilacak logo dosyalarini ekleyin.

## Onerilen dosya adlari

| Dosya Adi | Saglayici |
|-----------|-----------|
| `paytr.png` | PAYTR |
| `iyzico.png` | iyzico |
| `paynet.png` | Paynet |
| `craftgate.png` | Craftgate |
| `stripe.png` | Stripe |
| `bank-transfer.png` | Banka Havalesi / EFT |
| `cod.png` | Kapida Odeme |

## Onerilen format

- Format: PNG
- Arka plan: seffaf tercih edilir
- Boyut: 128x128 px veya 256x256 px
- Dosya boyutu: mumkunse 100 KB alti
