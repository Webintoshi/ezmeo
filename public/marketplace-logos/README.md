# Pazaryeri Logoları

Bu klasöre pazaryeri logolarını ekleyin:

| Dosya Adı | Platform | Boyut Önerisi |
|-----------|----------|---------------|
| `trendyol.png` | Trendyol | 128x128px veya 256x256px |
| `hepsiburada.png` | Hepsiburada | 128x128px veya 256x256px |
| `n11.png` | N11 | 128x128px veya 256x256px |
| `amazon-tr.png` | Amazon TR | 128x128px veya 256x256px |
| `ciceksepeti.png` | Çiçek Sepeti (opsiyonel) | 128x128px veya 256x256px |

## Format Önerileri
- **Format:** PNG (şeffaf arka plan tercihen)
- **Boyut:** 128x128px veya 256x256px
- **Dosya boyutu:** 50KB altında

Logolar otomatik olarak admin panelinde `/admin/markets` sayfasında görünecektir.
Eğer logo yüklenemezse, platforma özgü renklerde bir placeholder gösterilir.
