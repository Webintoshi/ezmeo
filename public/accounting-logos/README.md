# Muhasebe Programi Logolari

Bu klasore `/admin/muhasebe/fatura-entegrasyonu` sayfasinda kullanilacak logo dosyalarini ekleyin.

## Onerilen dosya adlari

| Dosya Adi | Saglayici |
|-----------|-----------|
| `parasut.png` | Parasut |
| `bizimhesap.png` | BizimHesap |
| `mikro.png` | Mikro |
| `logo-isbasi.png` | Logo Isbasi |
| `kolaybi.png` | KolayBi |
| `mukellef.png` | Mukellef |

## Onerilen format

- Format: PNG
- Arka plan: seffaf tercih edilir
- Boyut: 128x128 px veya 256x256 px
- Dosya boyutu: mumkunse 100 KB alti

Logo entegrasyonu yapilirken bu klasor referans alinabilir:

- `/accounting-logos/parasut.png`
- `/accounting-logos/bizimhesap.png`
- `/accounting-logos/mikro.png`
- `/accounting-logos/logo-isbasi.png`
- `/accounting-logos/kolaybi.png`
- `/accounting-logos/mukellef.png`
